Git是什么？
-----------

Git是开源的版本管理工具，用途与SVN、VSS相同，可以用来跟踪、记录你的代码工作，为个人的开发或团队协作提供便利（用习惯之后，你多半没法想象没有Git的日子了）。

Git和SVN相比，具有分布式的特点，而且速度更快（分支更轻量），仓库文件尺寸较小，所以受到很多人的推崇。
